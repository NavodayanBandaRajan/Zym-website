# Zym-website
Project-6
<br>
💪 IRON FITNESS GYM WEBSITE

A modern and fully responsive gym website created using HTML and CSS. This project is designed for fitness centers, gyms and personal trainers to showcase their services, membership plans, trainers and contact information in a professional way.

---
<br>
🚀 Features

- Responsive Design
- Modern Dark UI
- Smooth Scrolling Navigation
- Hero Section
- About Section
- Membership Plans
- Services Section
- Trainers Section
- Interactive Gallery
- Contact Form
- Google Map Integration
- Social Media Icons
- Professional Footer
- Hover Effects & Animations

---
<br>
🛠 Technologies Used

- HTML5
- CSS3
- Font Awesome Icons

---
<br>
📂 Website Sections
<br>
🏠 Home Section

A professional hero section with gym introduction and CTA buttons.

ℹ About Section

Information about the gym, mission and highlights.
<br>
💳 Membership Section

Different membership plans with pricing and features.
<br>
🏋 Services Section

Fitness services including:

- Weight Training
- Cardio Training
- Yoga Classes
- Diet Plans
- Personal Training
<br>
👨‍🏫 Trainers Section

Professional trainers with descriptions and specialties.
<br>
🖼 Gallery Section

Interactive gym image gallery with hover effects and lightbox preview.

📞 Contact Section

Contact information and user contact form.
<br>
🗺 Map Section

Google Maps integration for gym location.

📌 Footer

Quick links, social media icons and copyright section.
